# Mumbai Street Food Navigator - Development Context

## Project Overview
This is a comprehensive Mumbai Street Food Navigator web application built with React 18, TypeScript, and Vite. The application serves as a cultural guide to Mumbai's street food scene, incorporating local slang, neighborhood insights, and authentic food experiences.

## Key Features
- **Food Finder**: Advanced filtering and search for 20+ Mumbai street foods
- **Slang Translator**: 50+ Bambaiya slang terms with translations and context
- **Neighborhood Explorer**: 15+ Mumbai areas with food specialties and cultural insights
- **Cultural Integration**: Authentic Mumbai context throughout the application

## Development Guidelines
- Use TypeScript for all components with proper type definitions
- Implement responsive design with Tailwind CSS
- Use Lucide React for consistent iconography
- Follow React 18 best practices with hooks and functional components
- Maintain authentic Mumbai cultural context in all content
- Ensure accessibility compliance in all components

## Data Sources
All cultural data, food information, slang terms, and neighborhood details are sourced from the comprehensive `product.md` file at the project root. This file contains:
- 20+ detailed street food entries with prices, locations, and cultural significance
- 50+ Bambaiya slang terms with translations and usage examples
- 15+ neighborhood profiles with specialties and visiting information
- Cultural tips for authentic Mumbai street food experiences

## Code Standards
- Use semantic HTML elements
- Implement proper ARIA labels for accessibility
- Follow consistent naming conventions
- Add proper TypeScript interfaces for all data structures
- Use custom hooks for reusable logic
- Implement proper error handling and loading states
